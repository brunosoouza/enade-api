Estrutura da API

Todas as operações são feitas usando a rota: /questoes

metodo GET:
/questoes
Exibe todas as questões que estão salvas na api
/questoes/id
Exibe apenas um documento da coleção com base no seu id
metodo DELETE:
/questoes - parametro: id
Apaga um documento da coleção com base no seu id
metodo PUT:
/questoes - parametros: id, enunciado, resA, resB, resC, resD, correta
Atualiza um documento com os valores dos parametros passados
metodo POST:
/questoes - parametros: enunciado, resA, resB, resC, resD, correta
cria um novo documento com base os valores passados
Obs: uma operção feita com sucesso retorn sempre "feito"
